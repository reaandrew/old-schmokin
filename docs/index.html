schmokin

A wrapper for curl providing chainable assertions written in bash

curl -Ls https://github.com/reaandrew/schmokin/releases/download/0.1.0/schmokin_install | bash

Adds to existing curl capabilities

Any extra curl arguments can be placed after the -- separator.

./schmokin $URL --req-header "X-FU" --eq 'BAR' -- -H "X-FU: BAR"

Simple set of assertions

--eq
equals
--gt
greater than
--ge
greater than or equals
--lt
less than
--le
less than or equals
--contains
contains

Uses JQ under the hood for JSON extraction

./schmokin $URL --jq-expr '.status' --eq "UP"
